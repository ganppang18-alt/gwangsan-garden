왼쪽에서 정원 정보를 수정하면 오른쪽 미리보기에 바로 반영됩니다. 다 만든 데이터는 JSON으로 내려받아 나중에 20·50개소로 쉽게 확장할 수 있어요.

GitHub에 올린 사진은 cheom1.png처럼 파일명만 입력하면 휴대폰에서도 보입니다.

📊 엑셀 서식에 입력한 뒤 CSV UTF-8 형식으로 저장해 불러오세요. 불러오면 기존 데이터를 교체하니, 먼저 JSON 내보내기로 백업해두면 안전해요.

💡 수정하면 오른쪽 미리보기에 바로 반영되고, 자동으로 임시 저장돼요. 저장 버튼을 누르면 이 브라우저에 확실히 보관되어, 다음에 파일을 다시 열어도 그대로 남아 있습니다. 위도·경도는 임시 좌표라 정확한 값을 넣으면 지도·거리도 정확해집니다.
